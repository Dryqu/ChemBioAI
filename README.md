# ChemBio AI Insights

A responsive marketing site for **ChemBio AI Insights**, showcasing how artificial intelligence is accelerating breakthroughs across chemical and biological innovation.

## Preview locally

Open `index.html` in your browser. All assets are static, so no build step or tooling is required.

## Key features

- Responsive navigation with desktop and mobile layouts plus a Chinese language entry point.
- Hero narrative, search-enabled solution grid, industry spotlights, and analyst insight feed.
- Newsletter subscription panel with client-side confirmation messaging.

## Deployment

Deploy the repository root (containing `index.html`, `styles.css`, and `script.js`) to any static host such as GitHub Pages, Netlify, or Vercel.
