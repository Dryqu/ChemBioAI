# ChemBioAI
ChemBio AI Insights

## Run locally
1. Install a simple static server (for example `python -m http.server 8000`).
2. From the repo root, run the server and visit `http://localhost:8000`.

## Publish on GitHub Pages
1. Commit and push to the `main` branch.
2. In the repo settings, enable GitHub Pages with the `main` branch as the source (root folder).
3. Pages will serve `index.html` plus all content in `assets/`, `posts/`, and `category/` automatically.
